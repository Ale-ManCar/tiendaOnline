# Limitations

The project now has a NestJS API, PostgreSQL persistence, secure cookie authentication, password recovery, account password changes, server-backed carts, transactional checkout, stock decrementing, and server-backed public catalog reads. It is still not a fully automated ecommerce platform because real card payments, payment webhooks, inventory reservations before payment, and production observability are not complete.

GitHub Pages can host only the static storefront. The API and PostgreSQL database need separate production hosting before the site can operate as a real ecommerce business.
